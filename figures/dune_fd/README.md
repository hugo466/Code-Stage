# Organisation des figures DUNE FD

- `reference/` : reproductions de reference sans point ISS specifique.
- `flux/` : comparaisons de flux FD GLoBES/dk2nu.
- `detector_response/` : sections efficaces, efficacites et matrices de smearing FD.
- `iss23/` : figures ISS(2,3), donc cadre 3+1. Les points sont ranges comme `construct23_pointXX/`.
- `iss24/` : emplacement reserve aux figures ISS(2,4), donc cadre 3+2. Les points seront ranges comme `construct24_pointXX/`.

Convention pour les points :

```text
iss23/construct23_point70/fig4/
iss23/construct23_point70/probabilities/
iss24/construct24_pointXX/fig4/
```
